Stickmen Fighting Game

Stickmen Fighting Game is a competitive 2D browser-based fighting game developed using HTML, CSS, and JavaScript. The project was created to explore and understand real-time game mechanics, collision detection, animation systems, and player interaction.

The game focuses on smooth movement, responsive combat mechanics, and visually engaging sprite animations. Each character has directional attacks, dash abilities, blocking mechanics, knockback physics, and energy management systems.

Game Description

The game features two playable stickmen characters engaged in close-range combat. Players can move left and right, jump, punch, kick, block incoming attacks, and dash strategically to reposition themselves during fights.

A health and energy system adds tactical depth. Players must manage their energy carefully when performing advanced movements like dashing or combo attacks. Successful hits trigger knockback animations and temporary stun states.

Sprite-based animations were implemented to simulate realistic combat movement, including idle states, running, jumping, falling, attacking, taking damage, and death sequences.
